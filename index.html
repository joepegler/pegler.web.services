What I Build

Infrastructure that makes complex blockchain interactions reliable and shippable.

Transaction infrastructure

Bundlers, paymasters, retries, and simulation so user ops and swaps succeed under load. The system handles partial failures and latency without leaving reliability to chance.

Account abstraction systems

ERC-4337–compliant flows: gasless UX, session permissions, sponsored txs, and modern signing (passkeys, EIP-712/1271). Designed so product teams can ship onboarding and permissions without owning the plumbing.

Developer tooling

SDKs and APIs that abstract complex transaction flows so the next engineer can integrate and extend without tribal knowledge. Typed clients, clear errors, and docs that match behaviour.

Cross-chain execution

Route orchestration and execution across EVM and Solana: swap, bridge, and action providers with a single interface. Pre-execution validation and state handling so outcomes match expectations.
